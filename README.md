## my web3site

This website has 4 smart contracts:
 - Minting a zombie NFT collection 
 - Generate a random number to win an NFT (communication with chainlink VRF)
 - An NFT auction
 - Register comments
 
 Method & tools:
  - ehter.js
  - javaScript
  - html & css
  - react
  - solidity
  - openzeppelin
  - Chainlink VRF
  - ERC721 token standard
  - Ipfs distributed file storage
